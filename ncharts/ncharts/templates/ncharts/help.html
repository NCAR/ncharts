Help

  • 1-D Time Series Traces
    • To plot real-time data as it arrives, check the "Track Real Time" box. This option is only available for datasets whose end time is in the future. Your browser will poll the server for new data, at a time interval equal to 1/2 the average delta-T of the data. The plot will scroll and and start time will be updated when new data arrives.
    • When a time zone is selected the start time will be adjusted to indicate the same moment in time. The time labels on the plot will be in the selected time zone.
    • After changing a selection of variables, time or time zone, hit Plot to update the plot, as it is not automatically updated when a selection is made.
    • All selected variables with the same units will be plotted together.
    • The sequence of plots is alphabetical by units.
    • For all variables with the same units, the sequence in the legend will be alphabetical, by name.
    • Click on the variable name in the legend at the bottom of a plot to toggle the visibility of the trace, and dynamically rescale the remaining traces.
    • Use the sliders on the "navigator" trace below the chart to zoom in, and drag the selection to pan back and forth in time
  • Time X Height Plots (Currently only 449 profiler)
    • The plots will be in alphabetical order, by variable name.
    • For time X height plots, drag a rectangle across the plot to zoom in time.
    • Check "Track Real Time" to poll the server for real-time updates.

Bugs and Idiosyncracies

  • Response will be slow if you ask for a large quantity of data. Data reduction and compression are not supported.
  • Plots do not appear on the stock Android browser on phones. Use Chrome instead.
  • On phones, to scroll up or down , try swiping on the right-hand side to activate a small scrollbar.
  • Clicking Today in the datetime widget will set the time and date to the current time, in the browser's local time zone. It will then be interpreted in the selected time zone.
  • If you return to plot a previous dataset, you may have to do a browser refresh to see the variables and time you selected previously.
